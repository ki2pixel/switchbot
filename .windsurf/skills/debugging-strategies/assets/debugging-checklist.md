# Checklist de Débogage Rapide

- [ ] L'erreur est-elle reproductible en local ?
- [ ] Les variables d'environnement (`.env`) sont-elles correctes ?
- [ ] La base de données PostgreSQL est-elle accessible et les migrations sont-elles à jour ?
- [ ] Les dépendances (`venv`) correspondent-elles à celles de la production ?
- [ ] Y a-t-il une erreur CORS ou réseau dans la console du navigateur ?
