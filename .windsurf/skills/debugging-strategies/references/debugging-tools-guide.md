# Guide des Outils de Débogage

Ce guide recense les outils recommandés pour le débogage de l'application Flask et du backend Python.

## Outils Principaux
- **pytest** : Pour les tests unitaires et l'isolation des bugs.
- **pdb / ipdb** : Débogueurs interactifs pour Python.
- **Flask Debug Toolbar** : (Si activé en dev) pour analyser les requêtes, requêtes SQL et temps de rendu.
- **psycopg logging** : Pour tracer les requêtes vers PostgreSQL.
